Collective Predictive Coding for Distributed Bayesian Inference

How It Works

1
Propose

Each agent independently investigates and proposes an update w' to the shared knowledge.

2
Review

Agents are randomly paired. The reviewer scores consistency of w' vs wcurrent with their own findings.

3
Accept/Reject

Acceptance ratio: r = min(1, exp(logit(score') - logit(score))). If accepted, wcurrent updates.

The sample set {w[1], ..., w[I]} approximates the posterior q(w | o1, ..., oK).

Based on: Collective Predictive Coding as Model of Science (Taniguchi et al., 2024). arXiv:2409.00102
